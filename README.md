## Development

### Installation

```bash
$ git clone https://github.com/flekschas/enhancer-promoter-vis && enhancer-promoter-vis
$ npm install
```

### Commands

**Developmental server**: `npm start`
**Production build**: `npm run build`
**Demo build**: `npm run build-demo`
